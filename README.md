# Petpals-React-Native-Mobile
A React-Native based petadoption and gigworker connection hybrid app

Features:
Tinder Like Interface (Swipe) to adopt a pet
User Profile Creation/Updation
Pet Profile Creation/Updation
Matching Algorithm
Chat Feature
